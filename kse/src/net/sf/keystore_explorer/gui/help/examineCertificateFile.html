Examine a Certificate File

To examine a certificate file:
  1. From the Examine menu, choose Examine Certificate. Alternatively click on the Examine Certificate tool bar button:

        

  2. The Examine Certificate dialog will appear.

  3. Select the drive and folder where the certificate file is stored.

  4. Click on the required certificate file or type the filename into the File Name text box.

  5. Click on the Examine button.

  6. The Certificate Details dialog will appear.

  7. After viewing the details close the dialog by clicking on the OK button.
