Examine a CSR File

To examine a CSR file:
  1. From the Examine menu, choose Examine CSR. Alternatively click on the Examine CSR tool bar button:

        

  2. The Examine CSR dialog will appear.

  3. Select the drive and folder where the CSR file is stored.

  4. Click on the required CSR file or type the filename into the File Name text box.

  5. Click on the Examine button.

  6. The CSR Details dialog will appear.

  7. After viewing the details close the dialog by clicking on the OK button.
