===============================
 KeyStore Explorer Linux Setup
===============================

Prerequisites
-------------

KeyStore Explorer requires that an Oracle or OpenJDK 
Java Runtime Environment (JRE) be installed on your 
computer. KeyStore Explorer requires JRE version 1.6 
or greater.  The latest version of the Oracle JRE is
available free of charge from www.java.com.

The java executable must be added to your path.

Installing
----------

1) Extract the contents of this archive.

2) Run ./install.sh.

3) Follow the install steps.

4) KeyStore Explorer will be installed in the
   subdirectory kse5.0.
   
5) Change directory to 'kse5.0'.

6) Execute KeyStore Explorer by running: ./kse.sh
